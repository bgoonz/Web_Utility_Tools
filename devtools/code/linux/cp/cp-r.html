cp -R command

cp -R command in Linux/Unix.

cp -R command is used for recursive copy of all files and directories in source directory tree.

Syntax

$ cp -R srcdir destdir

Example

With verbose (-v):

$ cp -Rv dev bak
'dev/main.c' -> 'bak/dev/main.c'
'dev/test.c' -> 'bak/dev/test.c'
$
