How to view/merge text files

Linux view/combine text files.

View text file

$ cat list1.txt
milk
bread
apples

$ cat list2.txt
house
car

$

Combine text files

$ cat list1.txt list2.txt
milk
bread
apples

house
car

$

Combine 2 text files to another file

$ cat list1.txt list2.txt > todo.txt
$
