HSV to RGB conversion formula

When 0 ≤ H < 360, 0 ≤ S ≤ 1 and 0 ≤ V ≤ 1:

C = V × S

X = C × (1 - |(H / 60°) mod 2 - 1|)

m = V - C

(R,G,B) = ((R'+m)×255, (G'+m)×255, (B'+m)×255)
