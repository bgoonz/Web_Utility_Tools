How to type Infinity symbol on Facebook

Infinity symbol text typing on Facebook.

  1. Select with mouse the inifnity symbol below (∞).
  2. Copy the inifinity symbol below: press Ctrl+C keyboard keys or right click on mouse and select the Copy menu item.
  3. Paste in your facebook page: press Ctrl+V or right click on mouse and select the Paste menu item.
